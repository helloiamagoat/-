# GUI Calculator

Простой калькулятор с графическим интерфейсом на Python/Tkinter. Поддерживает базовые арифметические операции, очистку ввода и удаление последнего символа.

## Возможности
- Сумма, разность, умножение, деление, остаток и возведение в степень
- Быстрый возврат к предыдущему вводу с помощью клавиши Backspace (кнопка `⌫`)
- Безопасная обработка выражений через разбор AST без `eval`
- Окно фиксированного размера с тёмной темой

## Установка и запуск
1. Установите зависимости и установите проект в editable-режиме:
   ```bash
   pip install -e .
   ```
2. Запустите приложение:
   ```bash
   gui-calculator
   ```

Для проверки работоспособности без установки можно выполнить:
```bash
python -m calculator.app
```

## Сборка релиза
Соберите wheel/ sdist артефакты для публикации:
```bash
python -m build
```
Готовые файлы появятся в каталоге `dist/`.
